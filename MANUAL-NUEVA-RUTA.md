# Cómo añadir una ruta nueva a la web

Este contenido se ha trasladado a **`guia-gestion-rutas.html`**, con el paso a paso completo desde el navegador — añadir una ruta, añadir un tipo nuevo, eliminar una ruta, corregir datos a mano, textos, patrocinadores, redes sociales y colores. Para la referencia campo a campo de `rutas/config.json` y `rutas/theme.json`, ver `manual-configuracion.html`.
